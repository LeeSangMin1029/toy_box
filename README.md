# 1 - Follow Pointer
## UX
| Key | Description |
| --- | ----------- |
|  1  | 파티클의 수명 시간이 짧아진다. |
|  2  | 파티클의 수명 시간이 길어진다. |
| wheel | 마우스를 1px 움직일 때 생성되는 파티클의 개수가 1개씩 증가, 감소 |

위의 설명대로 Key에 해당하는 번호를 누르게 되면 속도가 느려지거나 빨라지는데 이는 현재 부분적으로 적용
